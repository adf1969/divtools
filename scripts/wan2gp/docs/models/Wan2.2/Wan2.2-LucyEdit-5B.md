# Wan2.2 Lucy Edit 5B

**Parameter Count:** 5 Billion  
**Model Type:** `lucy_edit`  
**Release:** Wan 2.2 series (2025)

## Overview

Lucy Edit 5B is a specialized instruction-guided video editing model designed for efficient, targeted modifications. Despite its smaller size (5B vs 14B), it delivers impressive editing capabilities with faster generation times.

## Best Use Cases

- **Quick Video Edits** - Fast turnaround for simple modifications
- **Style Adjustments** - Change colors, lighting, atmosphere
- **Object Manipulation** - Add, remove, or modify objects
- **Instruction-Guided Editing** - Natural language editing commands
- **Resource-Constrained Setups** - Lower VRAM requirements than 14B models

## Key Features

- **Efficient Architecture** - 5B parameters for faster generation
- **Instruction-Based Editing** - Natural language editing prompts
- **Sliding Window Support** - Edit long-form videos
- **Lower VRAM Requirements** - Suitable for mid-range GPUs
- **Video-to-Video Pipeline** - Direct video modification

- 8-12GB VRAM (typical)
