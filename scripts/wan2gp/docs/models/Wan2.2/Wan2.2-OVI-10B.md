# Wan2.2 OVI 10B

**Full Name:** Omni Video Intelligence 10B  
**Parameter Count:** 10 Billion  
**Model Type:** `ovi`  
**Release:** 2025

## Overview

OVI (Omni Video Intelligence) 10B is a specialized audio-visual generation model designed for creating videos of speaking characters with synchronized audio. It excels at lip-sync, facial animation, and natural speech-motion coordination.

## Best Use Cases

- **Talking Head Videos** - Create speaking character videos from audio
- **Dubbing/Localization** - Replace audio with lip-synced video
- **Virtual Presenters** - Generate AI presenters with natural speech
- **Character Dialogue** - Animate characters speaking with emotion
- **Educational Content** - Create instructional videos with narrators

## Key Features

- **Audio-Video Synthesis** - Generate video from audio input
- **Lip Sync** - Precise mouth movement synchronization
- **Facial Animation** - Natural expressions during speech
- **Multi-Speaker Support** - Handle dialogue between characters
- **Emotion Control** - Expressive facial movements

## Quick Reference

For complete technical details, see: [Wan2.2-ALL-MODELS-CATALOG.md](Wan2.2-ALL-MODELS-CATALOG.md)

**Default Settings:**
- Guidance Scale: 4.0
- FPS: 16
- Audio Sampling: 16kHz

**Key Parameters:**
- Audio Guide: Input audio file for lip sync
- Speaker Count: Number of speaking characters
- Emotion Tags: Control facial expressions

**LoRA Categories:**
- Character: 150-200+ LoRAs available
- Style: 100-150+ LoRAs available
- Camera/Util: 50-75+ LoRAs available
